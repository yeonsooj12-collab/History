# 사용자 안내: 수동 ChatGPT 흐름

이 프로젝트는 전용 GPT 또는 커스텀 GPT 사용을 전제로 하지 않는다. 현재 권장 흐름은 앱에서 일반 ChatGPT용 요청을 복사하고, ChatGPT가 반환한 JSON을 다시 앱에 붙여 넣는 방식이다.

## 흐름

1. 사용자 입력을 정리한다.
2. 일반 ChatGPT용 1차 요청을 복사한다.
3. `axis-finder` JSON을 앱에 적용한다.
4. 역사적 쟁점 하나를 선택한다.
5. 일반 ChatGPT용 2차 요청을 복사한다.
6. `editor` JSON을 앱에 적용한다.

## 현재 프롬프트 원칙

AI는 세계사 학습을 돕는 분석 편집자로 작동한다. 근거가 있는 사실만 사용하고, 불확실한 사건명·연도·제도명·작품명·사조명은 만들지 않는다. 입력과 닮은 역사적·문화적 패턴을 찾되, 차이와 한계도 함께 제시한다.
